# `webapp/frontend/src/domain` 目录说明

本目录用于放置前端领域纯函数：无 UI、无网络、无副作用，便于单测和复用。

## 文件职责

- `tags.ts`（本次新增）
  - `normalizeTag()`
  - `isAutoGeneratedTag()`
  - `isAutoTagForNovel()`
  - `scopeTagsForNovel()`

## 约定

- 只处理“规则和转换”，不直接读写组件状态。
- 不依赖浏览器 API（`window` / `localStorage` 等）。
- 需要副作用时放回 `composables/` 或组件层。

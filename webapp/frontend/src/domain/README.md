# `webapp/frontend/src/domain` Directory Guide

This directory contains frontend domain pure functions:
no UI, no network calls, and no side effects for easy testing and reuse.

## File Responsibilities

- `tags.ts` (new in this round)
  - `normalizeTag()`
  - `isAutoGeneratedTag()`
  - `isAutoTagForNovel()`
  - `scopeTagsForNovel()`

## Conventions

- Handle rules and transformations only; do not directly read/write component state.
- Do not depend on browser APIs (`window`, `localStorage`, etc.).
- If side effects are needed, keep them in `composables/` or component layers.

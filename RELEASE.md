# AI Novel Agent 1.0.0

面向作者的**长篇小说辅助写作**桌面应用（Windows）。用左侧 **Lore 设定**、中间 **任务与章节规划**、右侧 **流式生成与图谱**，把设定与连载状态串在一起。

---

## 系统要求

- **Windows 10 / 11，64 位**
- 可访问互联网（调用 **DeepSeek** 等大模型 API）
- 安装包体积约 **300MB+**，磁盘请预留足够空间

**无需安装 Python**，安装包已自带运行所需环境。

---

## 下载与安装

1. 在本仓库的 **Releases** 页面，下载 **`AI Novel Agent-1.0.0-Setup.exe`**（若版本号升级，以页面上的文件名为准）。
2. 双击运行安装程序，按向导操作；**可自行选择安装目录**。
3. 安装完成后，从开始菜单或桌面快捷方式启动 **AI Novel Agent**。

---

## 首次使用

1. 打开应用后，在界面右上角进入 **API 密钥**（或等价入口），填入你的 **DeepSeek API Key**。  
   - 密钥可在 DeepSeek 开放平台获取。  
   - 若使用第三方 OpenAI 兼容接口，请切换提供商并填写对应的 `API Key / Base URL / Model`。
2. 在 **lores** 对应目录放入你的设定 **Markdown**（子文件夹即标签），回到应用左侧勾选标签，即可在写作流程中使用。

---

## 你的文件保存在哪

默认情况下，数据与安装目录相邻，在 **`data`** 文件夹下，主要包括：

| 内容 | 说明 |
|------|------|
| **lores** | 你的设定文稿（`.md`） |
| **outputs** | 生成的正文等导出 |
| **novels** | 每本小说的数据库与存档 |
| **user_settings.json** | 界面里保存的 API 等设置 |

若安装到需要管理员权限的目录（如部分 **`Program Files`** 路径），写入可能受限；如遇保存失败，可改安装到当前用户有写权限的位置，或按系统提示以管理员运行。

---

## 本版本能做什么（摘要）

- 新建 / 管理小说，勾选 Lore 标签，**预览再运行**，减少误触消耗。
- **流式生成**章节与规划，可随时**中止**。
- **时间线与图谱**辅助把握人物与事件关系。

更完整的功能说明见仓库根目录 **README.md**。

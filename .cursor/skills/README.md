# Project Skills

This project provides custom Agent Skills tailored to the team's working style:

- `execute-plan-strictly`: strict attached-plan execution with ordered todo progression.
- `ui-button-and-theme-sweep`: exhaustive Vue button/theming consistency sweeps.
- `safe-large-file-refactor`: low-risk large-file decomposition into architecture layers.
- `refactor-doc-sync-and-regression`: documentation and regression closure after refactor.
